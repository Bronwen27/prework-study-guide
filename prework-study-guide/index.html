Prework Study Guide

HTML

  • HTML is like the foundation building blocks when building a house, this house can then be decorated using other code languages.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • A ul tag is an 'unordered list'
  • A li tag is an ordered list
  • Each section should relate to it's own topic or theme
  • HTML uses the id attribute as a way to identify elements and select them.

CSS

  • CSS is like painting and decorating a house that has already been built using HTML
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Assigning colors in CSS can be done in many ways, including by the name of a colour, or by its hexadecimal code, or hex #.
  • The link element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
